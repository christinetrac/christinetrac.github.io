Animal Crossing Tracker.
github
figma
An app for Animal Crossing™ players to track their friendships with villagers. Users input interactions with villagers so the app can determine the friendship levels. The purpose of this app is to help the user with the ultimate goal of receiving a villager's photo.
This is the first project I've ever created using React Native and Figma. I designed and developed this app by myself. I created this app because I loved playing Animal Crossing™ but the friendship mechanics behind the game were very unclear and many players were unsure how to increase their friendship levels with their villagers. After I read a document detailing all the complicated rules and interactions that impact friendship levels, I developed this app so users can easily track their friendship levels in a transparent manner. The app is still a work in progress and will be made available to the App Store once it is finished! My progress can be viewed in my Github repo and Figma mockups.
